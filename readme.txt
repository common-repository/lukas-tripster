This plugin use API Tripster.ru to use [shortcodes] for the show list of excursion.

== Description ==

With this plugin You can manually theming the output for list of excursions.

For correctly work on you hosting (PHP installation) must be enabled the cURL library.

== Installation ==

= Requirements =

To run Lukas Tripster plugin, we recommend your host supports:

* PHP version 5.3 or greater with installed library cURL

= Installation =

1. Upload the plugin files to the `/wp-content/plugins/lukas-tripster` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Tripster screen to configure the plugin


== Frequently Asked Questions ==

= How to use plugin? =

Just insert shortcode using the button of visual editor with Tripster logo or insert manually using [tripster city="Moscow" number="2"]
